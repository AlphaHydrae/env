# Environment

These are my configuration files. Use them at your own risk.

- Editor and keyboard configs.
- [Oh My ZSH](https://ohmyz.sh) prompt and
  [tmux](https://github.com/tmux/tmux/wiki) status bar:
  - Shared helpers live in [`bin/`](./bin/).
  - tmux behavior lives in [`tmux/`](./tmux/).
  - ZSH customization lives in [`oh-my-zsh/`](./oh-my-zsh/).
- [Nested tmux behavior](./docs/tmux-nested-behavior.md) with active/passive
  modes and visual indicators when connecting over SSH.
